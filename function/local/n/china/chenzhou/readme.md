中国车牌的规律很简单：**第一个汉字代表省份/直辖市，第一个字母代表该省份内的城市或地区**。

## 💡 两个你可能想知道的小知识

- **为什么有些城市有两个字母？** 当一个城市的汽车保有量迅速增长，原有的一个字母号段被用完时，就会启用新的字母。例如苏州除了**苏E**，还新增了**苏U**；成都除了**川A**，也启用了**川G**。

- **字母"I"和"O"为什么很少见？** 为了避免和数字"1"和"0"混淆，通常不在城市代码中使用字母"I"和"O"。不过，**"O"** 在一些省份会被用于警用或政府专用号牌。
